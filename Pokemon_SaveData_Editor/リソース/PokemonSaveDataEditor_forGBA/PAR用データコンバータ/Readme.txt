♪概要
　自作ツール「ポケモンセーブデータエディタ」は，
　ブートケーブルUSB（http://optimize.ath.cx/）等のいわゆる「マジコン」や，エミュ
　レータの吐き出す正規の128KBytesのデータを扱うことを前提としていたため，PARの作
　り出す前半64KBytesのセーブデータは編集できません．セーブデータエディタの方で対
　応できれば良かったのですが，製作時には全く眼中に無く，仕様変更は困難を極めるこ
　とが予想され，とりあえずはコンバータを作っておくことにしておきました．

♪注意
　・当たり前ですが，前半のデータ（セーブデータ一つと，ブロックデータ２つ）しか再
　現できません．セーブデータエディタで後半部分を編集してまたこのコンバータにかけ
　ると後半部分が消えてしまうのでご注意を．

　・また，改造コード製作機能もほとんど意味を成さなくなります．イベントのコードを
　サーチしたい場合は素直にブートケーブルを買って下さい．ROMも吸い出せていい感じ
　です．

　・改造コードのモードは"both"(新旧セーブデータ両方に書き込む)にしておく事をお勧
　めします．

♪起動までの準備について
　http://jp.sun.com/
　SunMicrosystemsのサイトから各自最新のJava 2 Platform
　Standard Editionをダウンロードしてインストールしてインスト
　ールしてください．その後OSによって起動方法が違います．

　　・Windows
　　同じフォルダにある「PAR_Converter.exe」をダブルクリック
　　して起動してください．

　　・その他のOS
　　シェルで「PAR_Converter.jar」のあるフォルダに移動して
　　java -jar PAR_Converter.jar
　　とすれば起動できます（多分）．

♪その他
　PAR用データコンバータはフリーソフトウェアです．
　（公共の福祉に反しない限り^^;）自由に使用できます．
　実行ファイルとドキュメントを書き換えず，完全な状態で配布する
　ならば，自由に転載，二次配布ができます．

　ただ転載時には事後でかまいませんのでメールで連絡をいただけ
　ると嬉しいです．
　またバグ報告，感想などをメールでいただけると嬉しいです．

　勿論，新機能の催促とかも待ってます^^
　できればどういう風な感じとかも書いてくれると嬉しいです．
