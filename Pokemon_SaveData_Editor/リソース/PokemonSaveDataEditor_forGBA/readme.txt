♪概要
　ポケモンGBA版のセーブデータを改変するためのエディタです．
　ポケモン一匹一匹を書き換えたり等，もっと上のレベルでのツール
　はあるのですが，こういった基本的なツールが無く不便だったため，
　今回製作しました．チートコード機能もあるので，一回作成すればあ
　とはボタン一つで改変が出来ます．また，コード作成を支援してくれる
　機能もあります．

 Ver1.36よりPAR用のデータコンバータを付けたので，PARユーザも利用
 できます．

♪注意
　こういったツールを使って楽をするのが嫌いな人も居ます．
　このツールで改変したセーブデータを用いる場合は対戦相手に許可を
　取りましょう．

♪起動までの準備について
　http://jp.sun.com/
　SunMicrosystemsのサイトから各自最新のJava 2 Platform
　Standard Editionをダウンロードしてインストールしてインスト
　ールしてください．その後OSによって起動方法が違います．

　　・Windows
　　同じフォルダにある「PokemonSaveDataEditor.exe」をダブルクリック
　　して起動してください．

　　・その他のOS
　　シェルで「PokemonSaveDataEditor.jar」のあるフォルダに移動して
　　java -jar PokemonSaveDataEditor.jar
　　とすれば起動できます（多分）．

♪使い方
　メニューを眺めてれば基本的なところは分かると思います．
　
　唯一分かりづらい気がする，セーブデータを右クリックしたときの
　「このセーブだけ一つのファイルに保存」の使い道を．
　　このソフトではそこまで高級なエディット機能をつける事はできません．
　そのため，外部のバイナリエディタを使ってデータのサーチするための機
　能です．
　
　一応言っておきますと，Windowsなら関連付けも出来ますよ．
　普通に使うよりもそっちのほうが便利です．

♪「チート作成機能」の仕様についてすこし
　差分をそのままとってコードを作っても，他人のセーブデータにそのままコードを適用
　してもうまくいかない可能性があります．比較してみて同じ場所でも，実はデータの一
　部かもしれないからです．
　そこで，「一行のバイト数で指定」で指定した数だけ同じデータが続いても，そこも
　データとしてコードの中に入れる仕様にしています．これで多分大丈夫だとは思います
　が，うまくいかない可能性もやはりあります．そのときは手動でコードを作ってくださ
　い．

♪その他
　ポケモンセーブデータエディタはフリーソフトウェアです．
　（公共の福祉に反しない限り^^;）自由に使用できます．
　実行ファイルとドキュメントを書き換えず，完全な状態で配布する
　ならば，自由に転載，二次配布ができます．

　ただ転載時には事後でかまいませんのでメールで連絡をいただけ
　ると嬉しいです．
　またバグ報告，感想などをメールでいただけると嬉しいです．

　勿論，新機能の催促とかも待ってます^^
　できればどういう風な感じとかも書いてくれると嬉しいです．

♪既知のバグ
　・たまに表示がおかしくなります．原因不明．再起動お願いします．
　・セーブ回数のカウントで，実際のデータと並びが異なる時があります．
　　多分実際のデータの方が仕様よりもズレてるんだと思いますが，一応バグ．
　　ズレててもちゃんとエラー無しに読み出せるので気にしないでください．
